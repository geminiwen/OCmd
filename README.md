#OCmd - 跨平台markdown编辑器

感谢 [@ghosert](http://www.weibo.com/ghosert) 开发了让我非常喜欢的Cmd 在线markdown编辑器 http://ghosertblog.github.io/mdeditor/
而我在这基础上开发的目的就是使用node-webkit的天然特性，使其具备导出HTML页面等功能。

<hr />

Feature:
 
1. 跨平台支持（node-webkit）
2. 支持HTML页面导出功能
3. 支持文本文件拖拽打开（UTF-8编码）
